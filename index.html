Monawbaty App

Monawbaty is an advanced scheduling and shift management application primarily developed for the Syrian Arab Red Crescent. Initially, it handled basic shift registrations, but subsequent versions introduced more robust features to streamline emergency response workflows. These enhancements included real-time data syncing, integrated notifications, and intuitive scheduling interfaces for paramedics.

This application leverages Flutter for a seamless cross-platform experience and uses various Firebase services—such as Firestore, Authentication, and Cloud Messaging—to deliver secure storage, user management, and instant updates. Bloc is used for state management, ensuring cleaner architecture and maintainable code.

Monasbat App

Monasbat is a simple yet effective event management application designed with Flutter. It was created for a local community in Saudi Arabia to handle various gatherings—like weddings, parties, and other celebrations—through a user-friendly interface. By integrating core Firebase services, Monasbat can offer real-time updates, invitation management, and basic analytics, making the coordination process smooth for both organizers and attendees.
